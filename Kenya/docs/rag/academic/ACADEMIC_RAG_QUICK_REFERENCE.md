## Carbon Revenue Projection — Derived Calculation

The following revenue projection is derived from the carbon density values above and should be cited as a calculated output, not as a standalone data source:

- **Lamu mangroves:** 560.23 Mg C ha⁻¹ x area (ha) x 3.67 (CO₂ conversion) x US$20/tCO₂e
- **Kwale mangroves:** 526.34 Mg C ha⁻¹ x area (ha) x 3.67 x US$20/tCO₂e
- **Combined projected annual revenue:** approximately US$3.59 million (at US$20/tCO₂e conservative price)
- **Carbon price sensitivity:** At US$50/tCO₂e the projection approaches US$9M/year; at US$10/tCO₂e it falls to ~US$1.8M/year
- **Primary citation needed:** Identify the survey report underpinning carbon density values (KMFRI, TARDA, or Go Blue project deliverable)
